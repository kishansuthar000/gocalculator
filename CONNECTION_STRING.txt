# MongoDB Connection - Quick Reference

## 📍 EXACT FILE LOCATION

```
d:\gocalc\backend\.env
```

Click this file and edit it.

---

## 📋 WHAT YOU'LL SEE

```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/gocalc  ← EDIT THIS LINE
JWT_SECRET=your_jwt_secret_key_change_in_production
NODE_ENV=development
```

---

## 🎯 PASTE YOUR CONNECTION STRING HERE

Replace the `MONGODB_URI` line with one of these:

### Local MongoDB (Easiest)
```env
MONGODB_URI=mongodb://localhost:27017/gocalc
```

### MongoDB Atlas (Cloud)
```env
MONGODB_URI=mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/gocalc?retryWrites=true&w=majority
```

---

## ✅ COMPLETE .env FILE

After editing, your file should look like:

**If using Local MongoDB:**
```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/gocalc
JWT_SECRET=your_jwt_secret_key_change_in_production
NODE_ENV=development
```

**If using MongoDB Atlas:**
```env
PORT=5000
MONGODB_URI=mongodb+srv://myusername:mypassword@cluster0.abc123.mongodb.net/gocalc?retryWrites=true&w=majority
JWT_SECRET=your_jwt_secret_key_change_in_production
NODE_ENV=development
```

---

## 🔑 GET YOUR ATLAS CONNECTION STRING

1. Go to: https://www.mongodb.com/cloud/atlas
2. Sign up → Create cluster
3. Click "Connect" → "Drivers"
4. Copy the connection string
5. Replace `username` and `password` with your DB credentials

---

## 💾 SAVE THE FILE

Press `Ctrl + S` to save after editing.

---

## ✨ THEN RUN

From `d:\gocalc\` folder:

```bash
npm start
```

Done! ✅
